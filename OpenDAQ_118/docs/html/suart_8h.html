suart.h File Reference

Description

header for suart.c Port and Pin setup here.

Set up the pins for the soft UARTs here.
Each software UART has an Rx and Tx pin. These can be assigned to any GPIO pins on the microcontroller.

Note
To re-assign pins:
In each section below, for each pin there are 3 places where the port letter needs to be changed, and 3 places for the pin number to be changed.
So for each pair of pins there will be 12 instances to be changed. Only the letter (ex: B) and the number (ex: 7) need to be changed. The reason for this is that there is both the soft uart and the corresponding direct connection (avalible to users in the cal menu) port and pin assignments done here.
